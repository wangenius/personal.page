title: VisibleBase
---

VisibleBase 是一个创新平台概念，用来重新定义早期创业项目的“发现”和“融资”机制。它的核心不是传统意义上的众筹或VC，而是一种「AI-Agent 驱动的、早期产品发现与微众支持平台」。

## 核心理念

VisibleBase 的出发点是——

> 让“早期信号”可见，让“微型投资”可行。

在传统模式中，早期产品往往在还没有流量、没有社交证明时就被埋没；而投资决策又被集中在少数VC手中。
VisibleBase 试图解决这两端的问题：

- 对创作者 / 创业者：提供一个「展示产品雏形 + 获取早期信任与支持」的舞台
- 对支持者 / 微型投资人：提供一个「用小额参与发现未来独角兽」的入口

## 背景

1. 风险投资体系的坍缩与重组：传统VC模式的结构性危机

- 资本集中：全球VC资金集中在顶级基金（a16z、Sequoia、SoftBank 等）手中，早期基金份额不断萎缩。
- 回报周期延长：过去的“十倍回报”周期被拉长到8-10年；AI时代更偏向赢家通吃，失败项目无残值。
- LP信任危机：欧美VC基金2023–2024年募资难度创十年新高，Pre-seed/Seed轮骤减。
- 退出困难：IPO窗口关闭、并购减少，资本流动性受限。

结果： VC不再承担“早期信号发现”职能，而在转向“成长阶段资产管理”。
VisibleBase 正是在这个空缺上切入：重建“早期发现层”。

2. 文化与认知背景：从“资本主义VC”到“社会化信号市场”

Kickstarter 实质是 预售平台 + 信任游戏，并非投资： 没有权益回报（reward only），依赖强文案和品牌信任，不形成结构化数据，项目死后无追踪价值。VisibleBase 本质是 “信号市场 + 智能体评估 + 可量化参与”，不以“预售产品”为主，而是以“预测未来”为核心：每个支持行为都是一个“市场信号”。 每个项目被 AI 结构化后进入「发现图谱」，投资与投票融合成“参与权重”。这种模式接近：信息市场（information market） × 预言机机制（oracle） × 众包VC”

3. Web3 / 区块链的作用

透明分配：智能合约可管理收益分配
可验证参与：每个 micro-backer 的参与记录上链
碎片化权益：Token化允许微额持份，不需要法律股权结构

4. 资本市场的转向：微投资与社群驱动资本

“社会化资本”正在成为趋势：Mirror、Zora 等创作者经济平台用 NFT 销售“叙事份额”；Crowdcube、Republic 等众筹平台，已支持“equity + token”混合模式；EigenLayer、Blast 等新经济协议在 web3 中引入“Restaking”概念，本质是流动性再配置。VisibleBase 是这种趋势在 “非金融化创新产品” 领域的版本。

资金形态的碎片化：过去：LP → GP → Founder；未来：个体 → 信号池 → 创作者。VisibleBase 以信号为媒介，将“流量”直接变成“资本”。

## 系统结构

VisibleBase 由三个层级构成：

1. Product Layer — 产品展示与进度透明层
   - 类似 GitHub 的项目页，展示核心假设、版本进度、roadmap
   - 支持 AI-Agent 生成产品摘要、竞争对比、里程碑推演

2. Signal Layer — 市场验证与社群信号层
   - 用户可以“收藏、评论、打赏、预测”等方式参与
   - 每个行为都成为可量化的「信号」，构成项目的热度曲线

3. Funding Layer — 微型投资与收益分配层
   - 支持代币化“支持额度”
   - 通过智能合约锁定收益比例（如未来营收分红、NFT-化回报、早期权益等）

## Agent 机制

VisibleBase 不只是一个展示平台，而是一个 AI-enhanced discovery engine。
Agent 的角色包括：

- Scout Agent（侦察代理）：自动挖掘互联网上潜在的创新产品
- Analyst Agent（分析代理）：总结产品潜力、评估信号、生成投资摘要
- Match Agent（撮合代理）：连接创作者与潜在支持者
- Crowd Agent（社群代理）：帮助社区筛选、评价与治理项目

## 与传统众筹 / 投资平台的区别

| 维度     | VisibleBase                  | 众筹平台 (Kickstarter等) | 投资平台 (AngelList等) |
| -------- | ---------------------------- | ------------------------ | ---------------------- |
| 主体     | AI驱动的发现引擎             | 创作者主导的展示         | 投资人主导的撮合       |
| 投资门槛 | 极低（支持 micro-backer）    | 低                       | 高                     |
| 透明度   | 产品信号链公开、AI分析可溯源 | 靠文案                   | 靠Pitch                |
| 激励模式 | Token化信号 + 收益份额       | 一次性回报               | 股权/份额              |
| 目标     | 提前捕捉趋势与潜力产品       | 预售                     | 投资回报               |

## 长远愿景

VisibleBase 是“后VC时代”的一种雏形实验：

> 用信号流 + 智能体取代 Pitch Deck + VC 决策。

未来，它可以演化为：

- AI Fund Layer：AI自动投资（Agent Fund）
- On-chain Credential Layer：基于参与记录形成“创作者信用画像”
- Creator Discovery Graph：全网创新项目的语义知识图谱

# 第五节课：前端框架（React）

**目标**：跳过传统 HTML/CSS/JS 的繁琐学习，直接上手业界最流行的前端框架 React，并做出你的第一个网页。

------

## 一、为什么要用框架？

1. **纯 HTML/CSS/JS 的问题**
   - 页面逻辑写在一堆文件里，难维护。
   - 重复代码多，缺乏结构化。
2. **框架的优势**
   - **组件化**：把页面拆分成可复用的“零件”。
   - **数据驱动**：界面自动跟随数据变化更新。
   - **社区生态**：大量现成组件 & 解决方案。

👉 小结：**框架让开发从“手工拼接”变成“模块化搭建”。**

------

## 二、React 的核心思想

- **组件化**：页面 = 组件的组合。
- **声明式 UI**：只描述“页面长什么样”，React 负责更新 DOM。
- **状态管理**：组件内部可以存储数据（state），当数据变了，UI 自动刷新。

👉 例子：

```jsx
function Hello() {
  return <h1>Hello, React!</h1>;
}
```

## 四、React 核心概念

1. **组件**

   - 函数组件：

   ```jsx
   function Button() {
     return <button>点我</button>;
   }
   ```

2. **Props（属性传递）**

   ```jsx
   function Hello(props) {
     return <h1>Hello, {props.name}</h1>;
   }
   
   <Hello name="Alice" />
   ```

3. **State（状态）**

   ```jsx
   import { useState } from "react";
   
   function Counter() {
     const [count, setCount] = useState(0);
     return (
       <div>
         <p>点击次数: {count}</p>
         <button onClick={() => setCount(count + 1)}>+1</button>
       </div>
     );
   }
   ```

## 五、成果练习：个人链接展示页

目标：做一个“个人主页”，展示头像、名字、社交媒体链接。

### 示例代码

```jsx
function Profile() {
  return (
    <div style={{ textAlign: "center", marginTop: "50px" }}>
      <img
        src="https://via.placeholder.com/150"
        alt="avatar"
        style={{ borderRadius: "50%" }}
      />
      <h1>你的名字</h1>
      <p>欢迎来到我的个人页面</p>
      <div>
        <a href="https://github.com/username" target="_blank">GitHub</a> | 
        <a href="https://twitter.com/username" target="_blank">Twitter</a> | 
        <a href="https://linkedin.com/in/username" target="_blank">LinkedIn</a>
      </div>
    </div>
  );
}
```

在 `App.jsx` 中引入：

```jsx
import Profile from "./Profile";

function App() {
  return <Profile />;
}
```

👉 学员就能快速做出一个网页名片。

## 六、部署上线

1. **推送代码到 GitHub**

   ```bash
   git init
   git add .
   git commit -m "My React profile page"
   git branch -M main
   git remote add origin https://github.com/username/my-app.git
   git push -u origin main
   ```

2. **用 Vercel 部署**

   - 登录 [Vercel](https://vercel.com/)
   - 连接 GitHub 仓库
   - 一键部署
   - 获取网址，分享给朋友

👉 学员的第一个“作品”就上线了。

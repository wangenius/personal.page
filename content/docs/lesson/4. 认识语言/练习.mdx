## 任务 3：定义一个类

创建 `user.js`：

```js
class User {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  introduce() {
    return `Hi, I'm ${this.name}, ${this.age} years old.`;
  }
}

const alice = new User("Alice", 25);
console.log(alice.introduce());
```

✅ 预期输出：

```
Hi, I'm Alice, 25 years old.
```

## 任务 5：最小 HTTP 服务器

创建 `server.js`：

```js
const http = require("http");

const server = http.createServer((req, res) => {
  if (req.url === "/hello") {
    res.end("Hello World");
  } else if (req.url === "/time") {
    res.end(new Date().toISOString());
  } else {
    res.end("Not Found");
  }
});

server.listen(3000, () => {
  console.log("服务器已启动: http://localhost:3000");
});
```

运行：

```bash
node server.js
```

✅ 预期结果：

- 打开 `http://localhost:3000/hello` → 显示 `Hello World`
- 打开 `http://localhost:3000/time` → 显示当前时间

## 任务 6（加分挑战）：返回 JSON

修改 `server.js`：

```js
if (req.url === "/user") {
  res.setHeader("Content-Type", "application/json");
  res.end(JSON.stringify({ name: "Alice", age: 25 }));
}
```

✅ 预期结果：

- 打开 `http://localhost:3000/user` → 返回：

```json
{"name":"Alice","age":25}
```

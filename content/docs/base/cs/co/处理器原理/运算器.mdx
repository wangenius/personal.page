# 运算器

## 运算器结构

1. 暂存寄存器 用户透明

2. 累加寄存器`ACC`

3. 通用寄存器组`Xs` 用户可见

4. 状态寄存器`PSW`用户可见

5. ALU

## ALU

{/* ![运算器](../../@asset/运算器.jpeg) */}

数字电路：

1. 组合逻辑电路：任意时刻的输出仅取决于该时刻的输入，与电路原来的状态无关

2. 时序逻辑电路：除了取决于输入，还取决于电路原有的状态。时序逻辑电路具有记忆，触发器，可以记录迁移可的输出状态，CPU就是一种复杂的时序逻辑电路

3. ALU是组合逻辑电路，使用时其输入端口A和B必须与锁存器相联，而且在运算过程中锁存器的内容是不变的，其输出也必须送至寄存器保存。

触发器：能存储一位信号的基本单元电路

锁存器：多位触发器，经常被用作运算器中的数据暂存器，临时存放数据的地方

ALU除了加减乘除、逻辑运算外，还具有先行进位逻辑。

**算术逻辑部件74181**：是4位ALU电路，4片74181组成一个16位全加器按4位一组的单重分组跳跃进位链，完成16种算数运算和16种逻辑运算。

串行运算器不存在先行进位。

C语言格式转换不会动位值。

# 设计模式

[参考](https://refactoringguru.cn/design-patterns/catalog)

## 创建型模式

这类模式提供创建对象的机制， 能够提升已有代码的灵活性和可复用性。

创建型模式包括：

- 工厂方法：Factory Method
- 抽象工厂：Abstract Factory
- 建造者：Builder
- 原型：Prototype
- 单例：Singleton

## 结构型模式

这类模式介绍如何将对象和类组装成较大的结构， 并同时保持结构的灵活和高效。

## 行为型模式

这类模式负责对象间的高效沟通和职责委派。

## 二重积分
| 性质 | 公式 |
| --- | --- |
| 区域面积 | $A=\iint_{D}d\sigma$|
| 可积有界 | $\iint_{D}f\left(x,y\right)\mathrm{d}\sigma=A\implies m\le f\left(x,y\right)\le M$ |
| 保号性 | $\left\|\iint_{D}f\left(x,y\right)\mathrm{d}\sigma\right\|\le\iint_{D_{}}\left\|f\left(x,y\right)\right\|\mathrm{d}\sigma$ |
| 估值定理 | $mA\le\iint_{D}f\left(x,y\right)d\sigma\le MA$ |
| 线性可拆 | $\iint_{D}\left\lbrack k_1f\left(x,y\right)+k_2g\left(x,y\right)]\mathrm{d}\sigma\right.=k_1\iint_{D}f\left(x,y\right)\mathrm{d}\sigma+k_2\iint_{D}g\left(x,y\right)\mathrm{d}\sigma$ |
| 区域可加 | $\iint_{D}f\left(x,y\right)\mathrm{d}\sigma=\iint_{D_1}f\left(x,y\right)\mathrm{d}\sigma+\iint_{D_2}f\left(x,y\right)\mathrm{d}\sigma$ |
| 中值定理 | $\iint_{D}f\left(x,y\right)\mathrm{d}\sigma=f\left(\xi,\eta\right)A$|
| 普通对称 | $\iint_{D}f\left(x,y\right)\mathrm{dxdy}=\left\{\begin{array}{}2\iint_{D_1}f\left(x,y\right)\mathrm{d}x\mathrm{d}y &f\left(-x,y\right)=f\left(x,y\right)\\ 0,& f(x,y)=-f(-x,y)\end{array}\right.$ |
| 轮换对称 | $\iint_{D}f\left(x,y\right)d\sigma=\iint_{D}f\left(y,x\right)d\sigma$ |


### 二重积分计算方法

直角坐标系：$\iint_{D}f\left(x,y\right)d\sigma=\int_{a}^{b}dx\int_{\varphi_1\left(x\right)}^{\varphi_2\left(x\right)}f\left(x,y\right)\mathrm{d}y=\int_{c}^{d}dy\int_{\psi_1\left(y\right)}^{\psi_2\left(y\right)}f\left(x,y\right)\mathrm{d}x$

极坐标系：$\iint_{D}f\left(x,y\right)d\sigma=\int_{\alpha}^{\beta}d\theta\int_{r_1\left(\theta\right)}^{r_2\left(\theta\right)}f\left(r\cos\theta,r\sin\theta\right)rdr$

## 三重积分

> 权重空间，可以把看作是密度$\rho$

| 性质 | 公式 |
| --- | --- |
| 物体体积 | $\iiint_{\Omega}dv=V$|
| 可积有界 | $\iiint_{\Omega}f\left(x,y,z\right)\mathrm{dv}=A\implies m\le f\left(x,y,z\right)\le M$ |
| 保号性 | $\left\|\iiint_{\Omega}f\left(x,y,z\right)\mathrm{dv}\right\|\le\iiint_{\Omega}\left\|f\left(x,y,z\right)\right\|\mathrm{dv}\quad \|\| \quad \iiint_\Omega f(x,y,z)dv \le\iiint_\Omega g(x,y,z)dv$ |
| 估值定理 | $mV\le\iiint_{\Omega}f\left(x,y,z\right)dv\le MV$ |
| 线性可拆 | $\iiint_{\Omega}\left\lbrack k_1f\left(x,y,z\right)+k_2g\left(x,y,z\right)]\mathrm{dv}\right.=k_1\iiint_{\Omega}f\left(x,y,z\right)\mathrm{dv}+k_2\iiint_{\Omega}g\left(x,y,z\right)\mathrm{dv}$ |
| 区域可加 | $\iiint_{\Omega}f\left(x,y,z\right)\mathrm{dv}=\iiint_{\Omega_1}f\left(x,y,z\right)\mathrm{dv}+\iiint_{\Omega_2}f\left(x,y,z\right)\mathrm{dv}$ |
| 中值定理 | $\iiint_{\Omega}f\left(x,y,z\right)\mathrm{dv}=f\left(\xi,\eta,\zeta\right)A$|
| 普通对称| $\iiint_{\Omega}f\left(x,y,z\right)\mathrm{dv}=\left\{\begin{array}{l}2\iiint_{\Omega_1}f\left(x,y,z\right)\mathrm{dv} & f\left(-x,y,z\right)=f\left(x,y,z\right)\\ 0,&f(x,y,z)=-f(-x,y,z)\end{array}\right.$ |
| 轮换对称 | $\iiint_{\Omega}f\left(x,y,z\right)\mathrm{d}v= \iiint_{\Omega}f\left(y,x,z\right)\mathrm{d}v$ |


### 三重积分计算方法

1. 直角坐标系
  1. 投影穿线法:$\iiint_{\Omega}f\left(x,y,z\right)dv=\iint_{D_{xy}}d\sigma\int_{z_1\left(x,y\right)}^{z_2\left(x,y\right)}f\left(x,y,z\right)dz$
  > 后积先定限，限内画条线，先交写下限，后交写上限
  2. 定限截面法:$\iiint_{\Omega}f\left(x,y,z\right)dv=\int_{a}^{b}dz\iint_{D_{z}}f\left(x,y,z\right)d\sigma$

2. 柱面坐标系:$\iiint_{\Omega}f\left(x,y,z\right)\mathrm{d}xdydz=\iiint_{\Omega}f\left(r\cos\theta,r\sin\theta,z\right)rdrd\theta dz$

3. 球面坐标系:$\iiint_{\Omega}f\left(x,y,z\right)\mathrm{d}xdydz=\iiint_{\Omega}f\left(r\sin\varphi\cos\theta,r\sin\varphi\cos\theta,r\cos\varphi\right)r^2\sin\varphi d\theta d\varphi dr$ */}

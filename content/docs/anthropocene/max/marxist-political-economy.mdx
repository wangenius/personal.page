## 劳动价值论（商品经济）

1. 目的：交换
2. 基础：生产资料私有制和个体劳动（简单商品经济），生产资料私有制和雇佣劳动（资本主义商品经济）
3. ​基本矛盾​：​私人劳动和社会劳动​（商品生产者的劳动两重性）的矛盾。交换是解决私人劳动和社会劳动之间矛盾的唯一途径。劳动的社会性质是由社会分工决定的，劳动的私人性质是由生产资料所有制决定的。在资本主义制度下这种矛盾进一步发展成为资本主义的基本矛盾，即生产社会化和生产资料资本主义私人占有之间的矛盾。
4. 解决基本矛盾的唯一途径：交换

### 劳动

1. 二重性：具体劳动和抽象劳动两方面，决定商品的二因素，是理解政治经济学的枢纽
2. 二重性矛盾的根源：社会劳动和私人劳动
3. 具体劳动：创造使用价值，反映人与自然的关系
4. 抽象劳动：形成商品价值，反映商品生产者的社会关系,劳动社会属性

### 价值

1. 二因素：使用价值和价值
2. 使用价值(自然属性)：构成社会财富的物质内容。源泉（不唯一）：具体劳动
3. 价值(社会属性)：凝结在商品中无差别的一般人类劳动，决定商品交换比例。即人的脑力和体力的耗费
   1. 唯一源泉：抽象劳动

4. 交换价值：由价值决定，使用价值是交换价值的物质承担者。商品的价值量与生产商品所耗费的社会必要劳动时间成正比，与劳动生产率成反比。
5. 关系：价值寓于使用价值之中，相互排斥，不可兼得
6. 货币：一般等价物。商品内在的使用价值和价值的矛盾发展成为外在的商品和货币之间的矛盾。
   1. 价值尺度
   2. 流通手段：及时交易
   3. 贮藏手段
   4. 支付手段：提前或延后交易，如租金或工资
   5. 世界货币

### 价值规律

1. 内容：商品的价值量由生产商品的社会必要劳动时间决定
2. 原则：等价交换
3. 基础：价值量
4. 表现形式：商品价格，围绕价值自发波动
5. 作用：​自发的调节生产资料和劳动力在社会各生产部门之间的分配比例，自发的刺激社会生产力的发展，自发的调节社会收入的分配。​价值规律在对经济活动进行自发调节时，会产生一些消极后果。导致社会资源浪费，阻碍技术的进步。导致收入两极分化。

## 剩余价值理论

由于唯物史观和剩余价值理论的发现，社会主义由空想变为科学。

### 资本主义制度

### 劳动力商品和资本形成

1. 货币资本化的​前提​：劳动力成为商品（自由人能够自由支配自己的劳动力，自由的一无所有），标志着简单商品生产发展到资本主义商品生产的新阶段。
2. 劳动力的使用价值：​是价值的源泉，在消费过程中创造新的价值。​一旦货币购买的劳动力带来了剩余价值货币也就变成了资本。资本家用货币购买的是雇佣工人的劳动力商品，劳动力商品的使用价值是劳动，劳动本身并不是商品，劳动力商品的使用价值具有成为价值源泉的特殊属性，从而创造了剩余价值。
3. 原始积累：是生产者和生产资料相分离，资本迅速集中于少数人手中。暴力手段剥夺农民的土地，用暴力手段剥夺货币财富。

### 资本主义制度

1. 经济制度的​产生​：以劳动力成为商品为前提，从小商品经济分化出来或从商人和高利贷者转化而来。
2. 政治上完成资产阶级革命，用资产阶级政权代替封建地主阶级政权。资产阶级统治的建立和资本主义生产方式支配地位的形成，标志着资本主义制度的最终确立。
3. 基本矛盾：生产社会化和生产资料资本主义私人占有之间的矛盾

​资本主义所有制：​资本家与工人的关系​不是完全占有，也不是人身依附​，而是​基于劳动者完全的人身自由基础上的“平等”关系。​资本家凭借对生产资料的占有，在等价交换原则的掩盖下，雇佣工人从事劳动无偿占有，雇佣工人创造的剩余价值。因此资本与雇佣劳动的关系具有了剥削与被剥削的对抗性质。资本主义所有制是雇佣劳动赖以存在的基础。是资本与雇佣劳动之间剥削与被剥削关系的体现。所有制是所有权的基础。

### 经济危机

1. 本质特征：生产相对过剩（相对于劳动人民有支付能力，而不是与劳动人民实际需要相比的绝对过剩）
2. 原因：根本原因是资本主义基本矛盾。形式上是货币作为支付手段和流通手段
3. 周期性：由资本主义基本矛盾运动的阶段性决定的。经济危机只能得到暂时缓解。资本主义经济危机周期性爆发的特点是社会资本再生产，也呈现出周期性的特点。这个阶段是危机，萧条，复苏和高涨。

### 资本主义生产过程

#### 一次生产过程

1. 基础：生产资料私有制和雇佣劳动
2. 目的和决定性动机，绝对规律：生产剩余价值
3. 劳动过程和价值增殖过程的统一
   1. 劳动过程：物质资料生产过程（转移价值）。三个基本要素：劳动者的劳动，劳动对象和劳动资料。
   2. 价值增值过程：价值增值过程是剩余价值的生产过程就是资本主义生产过程的主要方面，所谓价值增值过程是超过劳动力价值的，补偿这个一定的点而延长了价值形成过程。

4. 必要劳动：再生产劳动力价值的劳动（工资实际能买的劳动量）
5. 剩余劳动：无偿为资本家生产剩余价值，​是剩余价值的唯一源泉​。体现了资本家剥削的本质
6. 资本构成
   1. 有机构成：资本在资本主义生产过程中采取生产资料和劳动力两种形态。

7. 不变资本：生产资料形态，转移到新产品中
8. 可变资本：购买劳动力的资本
9. 相对过剩人口：有机构成提高，可变资本相对量减少，资本对劳动力的需求减少，不为资本价值增殖所需要

#### 资本循环周转和再生产

1. 追求剩余价值的结果：生产集中和资本集中
2. 资本扩大再生产：资本积累将剩余价值转化为资本，导致资本主义扩大再生产。​资本积累是资本主义扩大再生产的源泉​​。​​​资本积累的源泉是剩余价值​。资本积累规模的大小取决于资本家对工人的剥削程度。
3. 资本循环（个别资本的运动）：两个基本前提条件三种职能形式在空间上并存在时间上继起
   1. 购买阶段: 货币资本
   2. 生产阶段: 生产资本
   3. 售买阶段: 商品资本
   4. 资本周转的快慢：关键的因素有两个资本周转的时间和生产资本的固定资本和流动资本的构成。加快资本周转速度不能提高剩余价值率，但是能提高年剩余价值量

4. 固定资本：一定是不变资本，非消耗性支出
5. 流动资本：可变资本，消耗性支出
6. 社会再生产（社会总资本运动）：​核心问题是社会总产品的实现问题​，即社会总产品的价值补偿和实物补偿问题。客观上要求两大部类内部各个产业部门之间和两大部类之间保持一定的比例关系。
   1. 第1部类是由生产生产资料的部门所构成其产品进入生产消费领域
   2. 第2部类是由生产消费资料的部门所构成其产品进入生活消费领域
   3. 两大部类的生产都是在价值规律和剩余价值规律的作用下自发进行的，具有严重的盲目性，以至于社会总产品的实现，即实物替换和价值补偿难以顺利进行，最严重的就是引发经济危机。

#### 剩余价值规律

1. 剩余价值率: 
2. 绝对剩余价值：通过延长剩余劳动时间,加班
3. 相对剩余价值：通过缩短必要劳动时间,提高全社会技术水平和劳动生产率
4. 超额剩余价值：通过个别企业率先提高技术水平提高个别劳动生产率导致商品的个别价值低于社会价值时形成的差额,追逐超额剩余价值的结果是社会劳动生产率普遍提高,资本家普遍获得相对剩余价值。生产自动化并没有改变剩余价值的源泉。
5. 劳动生产率：劳动者生产商品使用价值的效率，单位劳动时间内生产商品数量的高低。单位商品所耗费的劳动时间。全社会劳动生产率的提高是资本主义加追逐超额剩余价值的结果。
6. 利润：本质剩余价值。资本家认为公司是劳动的价格或全部劳动报酬，因而掩盖了剥削，因为公司是必要劳动的产物。即仅仅是劳动力价值。平均利润是部门间及不同部门之间竞争形成的，超额利润是由部门内不同资本家之间的竞争形成的，二者并不冲突。表现: 看做是全部垫付资本的产物，掩盖了剩余价值和可变资本的关系资本家并不是把剩余价值看作可变资本的产物，而是把它看作全部垫付资本的产物，剩余价值便取得了利润的形态。
7. 平均利润：商品价值就转化为生产价格。价值规律的实现形式发生了变化。不同部门资本家按照等量资本获取等量利润的原则来瓜分剩余价值。平均利润率是不断下降的。剩余价值总量与社会总资本的比率
8. 工资：劳动力商品的价值，包括本人和家属的生存必须生活资料和本人的教育训练费用，不对应转移，工资表现为劳动的报酬，掩盖了剥削关系
9. 生产价格：利润转化为平均利润，价值也就转化为生产价格，生产价格是商品价值的转化形式，是生产成本与平均利润之和，商品不再以价值，而是以生产价格为基础进行交换，市场价格的变动不再以价值为中心，而以生产价格为中心。
10. 价值规律
11. 商品经济：价格围绕价值上下波动
12. 自由资本主义：价格围绕生产价格(成本(全部垫付资本)+平均利润)波动
13. 垄断资本主义：价格围绕垄断价格(成本+平均利润+垄断利润)波动

## 垄断资本主义

### 私人垄断资本主义

1. 基础：生产集中和资本集中
2. 资本集中最强有力的杠杆: 竞争和信用
3. 竞争：目的是获得垄断利润。没有停止,通过竞争继续维持垄断（垄断组织内部,组织之间,垄断和非垄断）
4. 垄断资本实质：获取垄断利润
5. 垄断价格：没有否定价值规律,是价值规律在垄断资本主义时期的表现,商品价格总额仍然等于商品价值总额
6. 世界扩展的形式：跨国公司

### 国家垄断资本主义

1. 存在: 基础设施和公共事业部门
2. 性质: 资本主义形式
3. 形式：短期价格协定，国际卡特尔，当代国家垄断资本主义的国际联盟（欧盟）
4. 体现: 总资本家剥削雇佣劳动者
5. 主要表现形式:
   1. 国家和私人共同投资开办合营企业
   2. 直接经营
   3. 通过多种形式参与私人垄断资本再生产
   4. 宏观调控(财政,货币)
   5. 微观规制(法律手段规范市场秩序)

### 金融垄断资本主义

1. 内涵: 工业垄断资本和银行垄断资本的融合
2. 壮大条件: 金融自由化和金融创新
3. 后果:
   1. 金融资本在国民生产总值和利润总额中所占越来越大
   2. 实体经济将利润投降金融,急剧膨胀
   3. 虚拟经济脱离实体经济
   4. 经济过度虚拟化,金融危机频繁发生

4. 形成途径
   1. 金融参与
   2. 资本参与
   3. 人事参与

5. 经济领域​参与制​: 股票控制
6. 国家机器控制
   1. 政府个人联合
   2. 议会代理人
   3. 政策影响: 政策咨询机构

7. 国际金融资本输出形式:
   1. 借贷资本
   2. 生产资本
   3. 商品资本

8. 国际金融资本垄断
   1. 金融垄断寡头化
   2. 金融垄断国际化
   3. 经济虚拟化
   4. 产业虚拟化

### 当代资本主义

当代资本主义最鲜明突出的特征是国际金融资本主义的垄断。

1. 经济全球化
   1. 根本动力：科技进步和生产力的发展
   2. 其他动力：跨国公司,各国经济体制变革和国际经济组织
   3. 表现：生产全球化,国际水平分工成为主导地位;贸易全球化,金融全球化

2. 所有制变化：随着股份公司成为主要的企业组织形式，私人股份资本所有制取代私人资本所有制成为占主导地位的所有制形式。国家资本所有制形成并发挥重要作用法人资本所有制崛起并成为居主导地位的资本所有制形式。
3. 劳资关系变化：职工参与决策，终身雇佣，职工持股。建立并实施了普及化全民化的社会福利制度。有一定改善社会阶层和阶级结构的变化。资本家: 不在直接经营管理企业。职业经理: 大公司经营活动的实际控制者
4. 制度变化
   1. 经济：改革有过一段时间的政府干预，普遍走上了强化市场调节，弱化政府干预的道路，去工业化和产业空心化日趋严重，产业竞争力下降，经济高度金融化，虚拟经济和实体经济严重脱节财政
   2. 政治：多元化,法治建设加强,改良主义政党对资本主义体制

5. 矛盾和冲突：经济发展失调，政治体制失灵，社会融合机制失效。极端思潮抬头，社会流动性退化，社会矛盾激化。

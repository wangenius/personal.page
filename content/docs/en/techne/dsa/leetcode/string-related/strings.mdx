- **Length of last word** – iterate from end skipping spaces, count letters until next space.
- **Binary addition** – two-pointer from right to left with carry; prepend leftover carry.
- **Roman to integer** – map symbols to values; subtract when a smaller value precedes a larger one.
- **First unique character** – frequency map then second pass to find first index with count 1.

Kadane’s algorithm (DP) runs in O(n): `dp[i] = max(nums[i], dp[i-1] + nums[i])`. Track the best answer while iterating.

```cpp
int maxSubArray(vector<int>& nums) {
    int best = nums[0], cur = nums[0];
    for (int i = 1; i < nums.size(); ++i) {
        cur = max(nums[i], cur + nums[i]);
        best = max(best, cur);
    }
    return best;
}
```

Brute force is O(n²); divide and conquer achieves O(n log n).

---
title: Subtree of Another Tree
slug: subtree-check
---

Two approaches:

1. **Serial match** – Serialize both trees via preorder traversal while inserting sentinels for null left/right children to avoid ambiguity. Then run string matching (e.g., KMP) to check if serialized `t` is a substring of serialized `s`.
2. **Recursive comparison** – Traverse `s`, and whenever you encounter a node whose value matches `t`’s root, invoke a helper that checks structural equality.

Serialization sample:

```cpp
void dfs(TreeNode* node, vector<int>& out) {
    if (!node) { out.push_back(nullVal); return; }
    out.push_back(node->val);
    dfs(node->left, out);
    dfs(node->right, out);
}
```

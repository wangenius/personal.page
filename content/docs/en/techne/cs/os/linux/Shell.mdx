---
title: Shell Basics
---

After logging into a text console you interact with the shell (commonly `bash`). Commands follow the pattern `command [options] arguments`. Notes:

- Options may be short (`-h`) or long (`--help`).
- Arguments are separated by spaces; use backslash `\` for line continuation.
- Shells are case-sensitive.

## Locale

View/update locale settings:

```bash
locale
LANG=en_US.UTF-8
export LC_ALL=en_US.UTF-8   # temporary
```

Edit `/etc/locale.conf` for persistent changes.

## Basic Commands

- `date`, `date +%Y/%m/%d`
- `cal` or `cal 2024`
- `bc` – interactive calculator

## Tab Completion

Press `Tab` to auto-complete commands, filenames, or options. Double `Tab` lists possibilities.

## Useful Shortcuts

- `Ctrl+C` – interrupt current command.
- `Ctrl+D` – EOF / exit shell.

## Process Helpers

- `lsof -i:3000` – find the process using port 3000.
- `kill -9 <pid>` – terminate.
- `nohup ./program &` – run in background immune to hangup.

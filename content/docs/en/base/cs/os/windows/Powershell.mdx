## Temporary Proxy

```powershell
$env:ALL_PROXY = "socks5://127.0.0.1:10808"
# Clear
$env:ALL_PROXY = ""
```

List current environment variables:

```powershell
ls env:*
```

## Basic Shell Commands

| Task | Windows | Unix |
| --- | --- | --- |
| List files | `dir` | `ls` |
| Change directory | `cd` | `cd` |
| Make directory | `md` | `mkdir` |
| Remove directory | `rd` | `rmdir` |
| Compare files | `fc` | `diff` |
| Copy files | `copy` | `cp` |
| Move files | `move` | `mv` |
| Rename | `ren` | `mv` |
| Delete | `del` | `rm` |

## Compilation

Use `g++ foo.cpp` (ensure the compiler path is on `PATH` in Windows). Common flags:

- `-o <file>` – Output executable name.
- `-g` – Include debug symbols (`gdb`).
- `-Wall` – Enable warnings.
- `-O1/-O2/-O3` – Optimization levels.
- `-DDEBUG` / `-UDEBUG` – Define/undefine macros.
- `-lm`, `-lgmp` – Link libraries (math, GMP). On Unix, math library requires `-lm`.

GNU Make can use built-in rules: `make foo` builds `foo` from `foo.c/.cpp`. Set extra flags via `export CFLAGS="..."` or `CPPFLAGS`.

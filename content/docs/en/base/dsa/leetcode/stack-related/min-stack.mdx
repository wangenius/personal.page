**Min Stack** – Store pairs `(value, currentMin)` so `getMin()` is O(1).

```cpp
stack<pair<int,int>> st;
void push(int x){
    int cur = st.empty()? x : min(x, st.top().second);
    st.push({x, cur});
}
void pop(){ st.pop(); }
int top(){ return st.top().first; }
int getMin(){ return st.top().second; }
```

**Monotonic Queue** – Maintain deque of candidates for window minimum. Each entry stores `(value, index)`. When pushing, pop from back while larger than new value; when popping, discard front if its index equals removed count.

Useful for sliding-window minimum/maximum in O(n).

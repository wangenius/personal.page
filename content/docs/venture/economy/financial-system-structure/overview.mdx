## 1. 金融体系在经济中的位置

前面的章节主要讨论了：

- 实体经济中的生产、消费、投资和贸易
- 宏观层面的产出、物价与失业

而金融体系回答的是另一个问题：

> 在时间和空间上，资金如何从“暂时不用的人”流向“需要使用的人”？

它承担的核心功能包括：

- 资金融通：储蓄与投资之间的桥梁
- 风险分担：通过保险、证券等工具分散和转移风险
- 支付清算：保障日常交易顺利完成
- 信息发现与价格形成：通过市场交易聚合分散信息

要看懂现代经济中的任何重大事件（危机、泡沫、监管改革），几乎都离不开这张金融体系的结构图。

## 2. 三个层次：机构、市场与基础设施

可以把金融体系粗略分为三个相互嵌套的层次：

1. 金融机构（who）
   - 银行、保险公司、基金、信托、券商、融资租赁公司等
   - 承担吸收储蓄、发放贷款、承保风险、资产管理等功能

2. 金融市场（where）
   - 货币市场、债券市场、股票市场、外汇市场、衍生品市场等
   - 提供价格发现和流动性平台

3. 金融基础设施（how）
   - 支付清算系统、中央证券登记结算、征信体系、交易系统
   - 保证交易“算得清、结得了、查得到”，并在压力时期支撑系统稳定

本模块聚焦的是第一和第三层：机构与基础设施，市场本身会在后续“金融市场”模块展开。

## 3. 核心角色概览

1. 中央银行
   - 发行基础货币
   - 充当银行体系的最后贷款人
   - 执行货币政策，维护金融稳定

2. 商业银行
   - 吸收存款、发放贷款
   - 提供支付结算服务
   - 在资产负债表内承担大量流动性与信用风险

3. 非银行金融机构
   - 保险公司：风险分担
   - 公募／私募基金：资产管理与资本市场投资
   - 信托、融资租赁等：提供多样化融资与资产配置工具

4. 影子银行与表外体系
   - 形式上不属于传统监管口径下的“银行”，但承担类似信贷与期限转换功能
   - 与正式银行体系通过资金和资产负债表高度联动

5. 金融基础设施
   - 支付系统：实时全额支付、零售支付网络
   - 清算与结算：中央对手方、中央证券登记机构
   - 征信与评级：信息采集与风险评估

## 4. 金融体系的结构性特征与风险传递

金融体系不是一堆孤立节点，而是一个高度互联的网络：

- 机构之间通过同业拆借、回购、衍生品和交叉持股相互关联
- 基础设施为这些关联提供“管道”和风险管理机制

这意味着：

- 有利的一面：风险可以在不同主体间分散和吸收
- 危险的一面：局部冲击可能通过网络迅速放大和扩散

2008 年全球金融危机、欧债危机等事件，都与金融体系内部结构和风险传递机制密切相关。

## 5. 本模块与后续模块的衔接

本模块重点回答：

- 金融体系有哪些主要“器官”（机构）和“血管”（基础设施）？
- 它们分别承担什么功能，又是如何互相连接的？

后续模块则在此基础上分别放大不同部分：

- 金融市场：具体资产（债券、股票、外汇、衍生品）如何定价与交易
- 监管与风险：宏观审慎、资本监管和压力测试等如何对冲系统性风险
- 危机与案例：把这些框架套到具体历史事件上进行复盘

掌握本模块的结构图，你在阅读任何关于“金融系统风险”“影子银行扩张”“支付系统创新”的报道时，都可以迅速定位：

> 这件事发生在金融体系的哪一层？涉及哪些关键机构与基础设施？可能通过哪些渠道向其他部分传导？

## 整体定位

《主权个人》（The Sovereign Individual）是詹姆斯·戴尔·戴维森（James Dale Davidson）和威廉·里斯-莫格勋爵（Lord William Rees-Mogg）在 1997 年出版的书，2020 年又再版。它被很多人称为“加密货币圣经”，因为书中在 90 年代就预言了数字货币、信息革命对社会结构的颠覆。

书的主旨：信息时代将彻底改变人类社会的组织形式，民族国家会逐渐失去权力，个人会获得前所未有的主权。

## 核心观点

1.  历史逻辑：暴力与社会形态
    - 人类社会经历了 狩猎采集 → 农业 → 工业 → 信息社会 四个阶段。

    - 信息革命降低了大规模暴力的收益，同时提高了个人知识和技能的价值。
2.  民族国家的衰落
    - 工业时代催生了民族国家，但在信息时代，国家征税与控制能力逐渐削弱。
    - 政府不再是不可替代的“保护者”，而可能被当作“服务机构”，个人会像“客户”一样选择是否与之合作。
3.  主权个人的崛起
    - 借助互联网、加密货币、去中心化技术，个体可以更好地保护财富，摆脱政府的高税与暴力机器。
    - “主权个人”指的是能在全球化数字经济中自立、自由交易、跨地域生活的精英个体。
4.  信息社会的经济逻辑
    - 财富的本质从土地、资本转向 知识与信息。
    - “思想即是财富”，认知精英和信息技能会成为最大杠杆。
    - 工作将被项目制、网络化替代，传统的职位和公司形态会弱化。
5.  未来的冲突与机遇
    - 民族主义、反动力量、新卢德分子可能会抵制变革。
    - 政府会尝试用暴力、法律、税收来延缓转型，但最终难以阻挡。
    - 信息革命带来的新社会组织，会像火药革命颠覆教会权力一样，重塑全球秩序。

- 农业革命（约公元前 10000 年）
  - 从游牧、狩猎采集 → 定居耕种。
  - 出现了土地、剩余粮食、财产的概念。
  - 暴力逻辑：谁能组织农田防御、征收剩余，就能统治。 → 出现酋长、领主、王国。
- 工业革命（18 世纪以后）
  - 蒸汽机、工厂、大规模生产。
  - 财富来源从土地 → 工厂/资本。
  - 暴力逻辑：需要大规模军队、火炮、海军来保护工业与殖民地。 → 出现民族国家、中央集权政府。
- 信息革命（20 世纪末至今）
  - 微处理器、互联网、加密技术。
  - 财富来源从资本 → 知识、信息、算法。
  - 暴力逻辑：大规模暴力回报下降，小规模个人防护/加密上升。 → 国家权力下滑，个人主权上升。

  暴力 = 长矛/石斧，基本上人与动物搏斗；组织形式是小部落。
- 阶段二：农业社会  
  暴力 = 土地的掠夺和保护，出现领主、地主；形成王权。

- 阶段四：信息社会  
  暴力 = 网络攻击、加密、防御性技术。大规模军队没用，知识与信息才是力量。 → 出现“主权个人”。

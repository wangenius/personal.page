在 macOS 上使用 **pem 证书**连接远程服务器（SSH 客户端）的方法如下：

## 1. 准备 PEM 证书

- 你应该已经从远程服务器（或云服务商，比如 AWS、阿里云、GCP）下载了一个 `.pem` 文件。

- 建议把证书文件放在 `~/.ssh/` 目录下，例如：

  ```bash
  mv ~/Downloads/your-key.pem ~/.ssh/
  ```

------

## 2. 修改证书权限

SSH 要求私钥文件必须有严格的权限，否则会拒绝使用：

```bash
chmod 400 ~/.ssh/your-key.pem
```

------

## 3. 使用 PEM 证书连接服务器

基本命令格式：

```bash
ssh -i ~/.ssh/your-key.pem username@remote_host
```

- `-i` 指定私钥文件路径
- `username` 是远程服务器用户名（常见的有 `ec2-user`, `ubuntu`, `root`）
- `remote_host` 是远程服务器的 IP 地址或者域名

```bash
ssh -i ~/.ssh/your-key.pem root@xx.xxx.xxx.xxx
```

------

## 4. 配置 SSH Config（可选）

如果你不想每次都写一长串命令，可以编辑 `~/.ssh/config` 文件：

```bash
nano ~/.ssh/config
```

添加：

```
Host myserver
    HostName 54.180.123.45
    User root
    IdentityFile ~/.ssh/your-key.pem
```

然后直接用：

```bash
ssh myserver
```

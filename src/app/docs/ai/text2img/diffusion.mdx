# Stable Diffusion 使用指南

## 简介

Stable Diffusion 是一个强大的 AI 文生图模型，能够根据文字描述生成高质量的图像。本指南将帮助您了解如何有效使用 Stable Diffusion。

## 基础使用

### 提示词（Prompt）编写

良好的提示词是生成理想图像的关键：

- 使用详细的描述词
- 按重要性顺序排列关键词
- 使用艺术风格相关词汇
- 可以添加权重，如 `(word:1.5)`

示例提示词：

```txt
一只可爱的橙色猫咪，坐在窗台上，阳光照射，细腻的毛发，温暖的氛围，8k超高清，细节丰富
```



### 负面提示词（Negative Prompt）

用于排除不想要的元素：
- 常用负面词：模糊、低质量、变形、多余的肢体等
- 建议始终包含基础负面提示词

## 重要参数说明

### 1. CFG Scale（提示词相关度）
- 范围：通常在 7-12 之间
- 较高的值会更严格遵循提示词
- 建议值：8

### 2. Steps（步数）
- 范围：20-50 步
- 步数越多，细节越丰富
- 建议值：30-35

### 3. Sampler（采样器）
常用采样器：
- Euler a：快速，质量适中
- DPM++ 2M Karras：较慢，质量较好
- DDIM：稳定，细节表现好

## 进阶技巧

### 1. 图像尺寸选择
- 标准尺寸：512x512, 768x768
- 横版：768x512
- 竖版：512x768

### 2. 种子值（Seed）
- 固定种子可以重现相同的图像
- 可用于微调和改进已生成的图像

### 3. 提示词权重设置

```txt
(高度重要的词:1.5) (次要词:1.2) (普通词:1.0)
```


## 常见问题解决

1. 图像质量不理想
   - 检查提示词是否足够具体
   - 适当增加步数
   - 调整 CFG Scale

2. 构图问题
   - 在提示词中明确指定构图
   - 使用适当的艺术风格关键词

3. 细节不足
   - 增加高清相关的关键词
   - 提高采样步数
   - 使用更大的图像尺寸

## 最佳实践建议

1. 保存成功的提示词配置
2. 记录有效的参数组合
3. 从简单开始，逐步调整
4. 多尝试不同的采样器
5. 建立个人的负面提示词库

#### 后端架构设计

- **Express 最佳实践**
  ```typescript
  import express from 'express'
  import cors from 'cors'
  import helmet from 'helmet'

  const app = express()

  // 中间件配置
  app.use(helmet())
  app.use(cors())
  app.use(express.json())

  // 路由分层
  app.use('/api/v1/users', userRouter)
  app.use('/api/v1/products', productRouter)

  // 统一错误处理
  app.use(errorHandler)
  ```

## 技术选型

### Web App

1. 前后端分离
	1. 前端：TypeScript + React + Vite + Zustand + Tailwind CSS + shadcn/ui	
	2. 后端：Node + Express
	3. 数据库：PostgreSQL
2. 全栈
	1. Next.js
	2. 数据库：PostgreSQL

### 小程序

1. Taro + React
2. 数据库：PostgreSQL

### 桌面应用

1. Electron:使用[electron-vite](https://cn.electron-vite.org/guide/)框架
2. Tauri:使用[tauri](https://tauri.app/v1/guides/getting-started/setup/vite)框架
3. 数据库：[PostgreSQL](https://neon.tech/postgresql/postgresql-getting-started)

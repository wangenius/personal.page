# WanGenius 介绍

WanGenius 是一个基于 Next.js 构建的现代化文档和博客系统，它具有以下特点：

## 主要特性

### 1. Markdown 支持
- 完整的 Markdown 语法支持
- MDX 支持，可以在 Markdown 中使用 React 组件
- 代码高亮和复制功能

### 2. 主题定制
- 内置亮色和暗色主题
- 可自定义主题颜色
- 响应式设计

### 3. 性能优化
- 基于 Next.js 的静态生成
- 自动图片优化
- 增量静态再生成

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/wangenius/wangenius.git
```

2. 安装依赖
```bash
pnpm install
```

3. 启动开发服务器
```bash
pnpm dev
```
